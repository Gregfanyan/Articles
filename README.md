## Setup

Feel free to clone the app, install with the command `npm install` or `yarn install ` and run `npm start` or `yarn run`

## Features

- Fetch web api
- Switch to two different API URL
- Load More articles 
- Mobile-first Design
- Every artilce link redirects to the current post
